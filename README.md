Goinmarketcap
=============

Super simple cryptocurrency price ticker that displays market data from Coinmarketcap

Usage:
```
go build
go run main.go
```

Example output:
```
coin     price
BTC    8191.11
ETH     365.91
BCH    1196.53
XRP       0.24
DASH    498.72
LTC      70.88
MIOTA     0.91
NEO      34.78
XMR     139.32
XEM       0.21
```
